# AI Drone Swarm for Disaster Relief

Final year project — University of KwaZulu-Natal, 2026

Five AI models that work together as a drone swarm system for disaster relief:
- M1: SwarmNet — drone task allocation using Graph Neural Networks
- M2: SurvivorNet — aerial person detection using YOLOv8
- M3: PathRL — autonomous drone flight using PPO reinforcement learning
- M4: SupplyOpt — delivery route optimisation using MARL
- M5: DamageMap — terrain damage classification using Random Forest

All models built and tested in Google Colab using free tools.
